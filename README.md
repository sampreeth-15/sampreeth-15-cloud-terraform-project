# Cloud Infrastructure with Terraform (Azure DevOps Project)

## 🚀 Overview
This project demonstrates an end-to-end DevOps pipeline using Terraform, Azure, Kubernetes, and CI/CD.

## 🛠 Tech Stack
- Terraform (IaC)
- Microsoft Azure
- Azure DevOps (CI/CD)
- Kubernetes (AKS)
- ArgoCD (GitOps)
